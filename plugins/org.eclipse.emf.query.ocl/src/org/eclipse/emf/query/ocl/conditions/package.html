Object Constraint Language extensions to the EMF Query Conditions framework.

Package Specification

This package defines the basic OCL-based conditions that are used to formulate OCL queries over EMF models. {@link org.eclipse.emf.query.ocl.conditions.OCLCondition}s apply OCL expressions to {@link org.eclipse.emf.ecore.EObject}s. It allows the integration of OCL expressions with other {@link org.eclipse.emf.query.conditions.eobjects.EObjectCondition}s.
