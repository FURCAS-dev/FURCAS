Object Constraint Language query/constraint evaluation API.

Package Specification

This package defines the API for constructing, validating, and evaluating OCL queries and constraints on EMF model elements. The syntax of queries is implemented by the {@linkplain org.eclipse.emf.ocl.expressions OCL expressions} API.

@deprecated Use the equivalent API in the {@link org.eclipse.ocl} package, instead, in particular the {@link org.eclipse.ocl.OCL} to create {@link org.eclipse.ocl.Query} objects. @see org.eclipse.emf.ocl.expressions
