Abstract API for OCL expression syntax.

Package Specification

This package defines the abstract API for the representation of OCL {@linkplain org.eclipse.emf.ocl.expressions.OCLExpression expression} syntax. OCL expressions are used to implement OCL {@linkplain org.eclipse.emf.ocl.query.Query queries} and constraints.

For walking an OCL expression AST, see the {@link org.eclipse.emf.ocl.expressions.Visitor} interface. For other utilities (such as the EMF switch), see the {@link org.eclipse.emf.ocl.expressions.util} package.
