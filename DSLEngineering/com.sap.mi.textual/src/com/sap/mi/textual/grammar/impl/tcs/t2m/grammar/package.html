Contains the toolset for writing an ANTLR3 grammar for a given TCS Syntax model.
ANTLRGrammarGenerator is the main controlling class in this package. It represents a facade to the classes in this package. ANTLR3GrammarWriter deals with creating a bytestream suitable as ANTLR3 grammar. All other classes are created and being invoked by ANTLRGrammarGenerator, and will use an instance of ANTLR3GrammarWriter to write their outputs.
