Implementation of the Ecore binding of the the Types package of the OCL Abstract Syntax.

Package Specification

This package implements the Ecore binding for the Types package of the OCL Abstract Syntax, as specified by the interfaces in the {@link org.eclipse.ocl.ecore} package. The types in this package are not typically used by clients. Rather, instances of the abstract syntax classes are created either using the {@linkplain org.eclipse.ocl.ecore.EcoreFactory factory interface} or by parsing OCL text.
