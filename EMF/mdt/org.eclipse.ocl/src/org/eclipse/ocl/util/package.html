Miscellaneous utilities in support of the OCL parser and environment implementations.

Package Specification

This package defines a number of utilities that support the implementation of the OCL parser and of metamodel bindings (i.e., {@link org.eclipse.ocl.Environment} implementations). These utilities include:

  • a {@link org.eclipse.ocl.util.Bag bag} collection definition
  • utility classes for manipulation of OCL {@linkplain org.eclipse.ocl.util.CollectionUtil collections} and {@linkplain org.eclipse.ocl.util.OCLStandardLibraryUtil standard library types}
  • a {@linkplain org.eclipse.ocl.util.ToStringVisitor visitor} transforming OCL expressions to strings (as a debugging aid)
  • an interface for manipulation of {@linkplain org.eclipse.ocl.util.Tuple tuple values}
  • utilities for working with {@linkplain org.eclipse.ocl.util.UnicodeSupport UNICODE} text
@see org.eclipse.ocl
